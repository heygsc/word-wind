# Word Wind

在线背单词网页🎨

## 点击链接，即可访问网页版：[https://word-wind.pages.dev](https://word-wind.pages.dev)🧭

## 欢迎点点star⭐

## 欢迎关注[作者](https://github.com/heygsc)

这个项目的目的，就是为了做一个丝滑的背单词页面，享受于此。

单词顺序为舒服的乱序，涵盖了初中、高中、大学四六级、考研、托福和SAT，满足各类人群需求。

包含音标以及读音播放、释义、例句、词组。

支持主题切换，多端适配。

## 如何使用

这个网站的使用非常简单。

可以选择你希望背的单词级别，页面会展示当前单词属于什么级别，以及单词的序号。

目前不够熟练的，选“不会”按钮。

点击“不会”按钮，就是不会的单词。如果某个不会的单词已经熟练，可以直接点击X按钮进行删除。
